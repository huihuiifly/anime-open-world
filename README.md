# 幻境旅人 · 二次元开放世界 Demo

一个**单文件、打开即玩**的二次元风格第三人称开放世界原型，基于 Three.js（纯网页，无需安装）。

## 运行方式

**方式一（最简单）**：直接双击 `index.html` 用浏览器打开即可。

**方式二（推荐，避免个别浏览器限制）**：在本目录起一个本地服务器再访问：

```bash
python3 -m http.server 8000
# 然后浏览器打开 http://localhost:8000
```

进入后点击「点击进入世界」锁定鼠标即可游玩。

## 操作

| 按键 | 功能 |
|------|------|
| `W A S D` | 移动（相对镜头方向）|
| `Shift` | 奔跑（消耗体力）|
| `Space` | 跳跃 |
| `鼠标移动` | 自由转动视角（yaw + pitch）|
| `滚轮` | 拉近 / 拉远镜头 |
| `左键` | 连击攻击（3 段连招）/ 砍树 |
| `F` | 靠近岩壁时攀爬 |
| `B` | 进入/退出建造模式 |
| `E` | 建造模式下放置房屋（消耗 8 木材）|
| `R` | 建造模式下旋转预览 |
| `T` | 靠近城镇 NPC 时对话 |
| `Esc` | 解锁鼠标 |

## 已实现的功能

- **第三人称 + 自由视角**：鼠标控制环绕镜头，滚轮缩放，镜头平滑跟随并避免穿地。
- **精细二次元女性角色**：正常人体比例、可见脖颈、自然斜肩、五指手、关节包覆无突骨；分明发束短刘海（止于眉上）+ 垂在脑后的细双马尾、细化的双色虹膜大眼（含上眼睑线/双高光）、闭合唇线、腮红；带腰带的**百褶裙** + 双层裙摆 + 蓝饰边、长筒袜、手持短剑。
- **动作模组（状态机）**：待机 / 行走 / 奔跑 / 跳跃 / 下落 / 攀爬，程序化关节动画带阻尼过渡，待机放松弯肘、重心转移，不僵硬。
- **三段连招**：左键可衔接 1→2→3 段不同挥砍动作（横斩 / 反斩 / 劈旋），HUD 显示连击数。
- **布料与头发动态**：奔跑/行走/转身时裙摆与双马尾会随动作惯性摆动甩动。
- **常驻生理动作**：呼吸起伏、自动随机眨眼、待机微晃、手腕跟随。
- **砍树**：树有耐久，攻击命中会摇晃，砍倒后倒下并产出木材。
- **简易建造**：`B` 进入建造模式，角色前方出现半透明房屋预览，`R` 旋转、`E`/左键花费木材放置（建成的房屋带碰撞箱）。
- **敌人（史莱姆 · 被动）**：世界各处随处刷新、闲逛弹跳；**只有被你攻击后才会反击**追打，头顶血条，击败得经验，稍后在别处重生。
- **建筑/树木/岩石碰撞箱**：角色会被挡住、推开，不再穿模进入。
- **昼夜循环（真实节奏）**：现实 5 秒 = 游戏 1 分钟，24 小时制，整天约 2 小时；太阳/月亮绕行，天空、雾气、光照在白天/黄昏/夜晚间过渡，夜晚出现星空月光，HUD 显示时钟。
- **自然环境与精致城镇**：分层着色起伏地形 + 概率生成的山丘、铺满草丛的地面（不再光滑）、团状树冠树木、岩石、灌木、可攀爬岩壁；房屋含地基、转角木柱、横梁、出檐、烟囱、门框、十字窗格与台阶，中央喷泉。
- **城镇 NPC**：村长/店员/铁匠/孩子/旅商等 5 位村民，会面向你、头顶有标识；靠近按 `T` 可对话（多句循环）。
- **角色数据**：血条 / 体力条 / 经验条 / 等级 / 木材 / 击败数。

## 当前限制（原型范围）

- 美术资源均为**程序化几何体生成**，非高精度建模/贴图。要再精致可导入 `.glb`/`.vrm` 角色模型与 PBR 贴图。
- 角色与敌人碰撞、复杂地形碰撞（斜坡/台阶）做了简化（圆形近似）。
